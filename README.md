Library Management System (LMS)

Project Description:
This project implements a Library Management System (LMS) where users can manage a collection of books. It provides functionality to add, edit, delete, search, and view books.

Features:
Add a Book: Enter Book ID, Name, and Author.
Edit a Book: Update details of a book using its Book ID.
Delete a Book: Remove a book using its Book ID.
Search for a Book: Find books by ID, Name, or Author.
View All Books: Displays all books in a formatted way.
View Statistics: Displays the total number of books.
